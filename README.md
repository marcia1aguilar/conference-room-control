## Conference Room Control System - Touch Panel Prototype


This technical prototype is a high-fidelity frontend designed for corporate conference room touch panels. It focuses on simplifying the hybrid meeting experience through an intuitive and modern user interface.

## ✨ Key Features

* Hybrid Meeting Focus: Dedicated controls for screen sharing and toggling in-room cameras to improve remote participant engagement.

* Touch-Optimized UI: Designed with high-contrast elements and fluid interactions specifically for touch-screen hardware.

* Responsive Architecture: Built with a "design-first" approach inspired by modern AV control standards (Crestron, Extron, Q-SYS).

* Clean Code: Strongly typed components using TypeScript for better maintainability and fewer runtime errors.

## 📝 Design Process
This project started as a design concept in Figma, aiming to bridge the gap between traditional AV hardware controls and modern web-based UI standards. The goal was to translate complicated AV configurations into a clean, modern dashboard, visually polished experience for the end-user.
